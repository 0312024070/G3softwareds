# メール送信設定

このシステムでは、以下の2種類のメールを送信できます。

1. 新規登録時の6桁確認コード
2. ログイン時の3つの絵文字認証コード

## 1. `.env` ファイルで設定する方法

`emoji_auth_system` フォルダ内の `.env.example` をコピーして、`.env` という名前のファイルを作成します。

```env
SMTP_SERVER=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=送信用Gmailアドレス
SMTP_PASSWORD=Googleで作成した16桁のアプリパスワード
SMTP_FROM=送信用Gmailアドレス
```

Gmailを使う場合、通常のログインパスワードではなく、Googleアカウントで発行するアプリパスワードを使います。

## 2. 起動する

```powershell
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
.\.venv\Scripts\python.exe app.py
```

## 3. 新規登録の確認

新規登録画面でメールアドレスを入力して送信すると、そのメールアドレスへ6桁の確認コードが送信されます。
確認コードを入力して正しければ、アカウントが作成されます。

## 4. ログイン認証の確認

登録済みユーザーでログインすると、登録メールアドレスへ3つの絵文字認証コードが送信されます。

## 注意

- `.env` は絶対にGitHubへアップロードしないでください。
- アプリパスワードやGmailのパスワードをソースコードに直接書かないでください。
- SMTP設定がない場合や送信に失敗した場合は、授業用確認のためターミナルにもコードを表示します。
